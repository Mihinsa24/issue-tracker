# Issue Tracker Web App

A full-stack Issue Tracking web application built using **React (Frontend)** and **Node.js + Express + MongoDB (Backend)**.  
It supports full CRUD operations for managing issues.

---

##  Features

- Create new issues
- View all issues
- View single issue details
- Update issues
- Delete issues
- Authentication (Login/Register)
- Search and filter issues
- Status tracking (Open / In Progress / Closed)

---

##  Tech Stack

### Frontend
- React (Vite)
- Axios
- Zustand (State Management)
- CSS

### Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- bcrypt
